# SCOUTMASTER

## Technology Stack
- PHP
- Node.js
- HTML
- CSS

## Features
- **Badge Progress Tracking**: Users can track their badge progress in real-time.
- **Registration Forms**: A user-friendly interface for scouts to register.
- **Virtual Meeting API**: Conduct virtual meetings seamlessly.
- **Audit Logs**: Detailed logs for tracking actions and changes.
- **Scout and School Management**: Manage scout units and school details efficiently.

## User Roles and Permissions
### Scout
- View badge progress
- Register for camps

### Scout Leader
- Manage scout registrations
- Track badge progress and provide feedback

### Admin
- Full access to manage scouts, leaders, and schools
- Generate reports and audit logs
